# Visual & Prop Lexicon

A library of recurring objects that should appear, evolve, and reappear across the body of work. By year three, the audience should be able to recognize a Jevoy Palmer piece by its objects alone.

## Script-Specific Props

Props assigned to specific themes in the Core 12:

| Script | Props | Visual Demonstration |
|--------|-------|---------------------|
| 1: Camera Change | Mirror, phone on tripod, dinner table | Mirror vs. phone side-by-side on table — "one forgets, one remembers" |
| 2: Avoid Visibility | Scale, invoice, cloth-covered product | Put "fear of judgment" on one side, invisible costs on the other |
| 3: One Clip | Photo frame, single puzzle piece, evidence board | Hold up one photo alone, then surround it with many — context changes meaning |
| 4: Context | Cropped photo, torn page, picture frame | Show cropped image that looks suspicious, then reveal full image |
| 5: Control Narrative | Blank canvas, empty frame, whiteboard | Hold empty frame — "If you don't put anything in, people will" |
| 6: Direction Not Content | Compass, map, scattered sticky notes | Throw sticky notes on table, then arrange around one central question |
| 7: Seen Not Exposed | Window blinds, curtain, spotlight | Adjust blinds — "This is not hiding. This is not always wisdom either. This is visibility with boundaries." |
| 8: Clarity Over Creativity | Fogged glass, clean lens, tangled cord | Show foggy lens, wipe clean — "Creativity gets attention. Clarity gives people sight." |
| 9: Silence Assumptions | Empty chair, blank name tag, unclaimed package | Show empty chair in a meeting — "Nobody thinks the chair has no meaning" |
| 10: Understand Faster | Blurry photo vs. focused photo, labeled box | Hold blurry image, then focused version — "This is what specificity does" |
| 11: Strategy vs. Posting | Notebook, calendar, unpublished draft, launch button | Flip through pages of plans, show zero published posts |
| 12: Authority Without Viral | Trust jar, coins, bright flash, tree rings | Drop coins into jar one by one — "This is authority." Flash bright light — "This is virality." The jar remains. |

## Prop Rules

1. **Props should not decorate — they should argue.** Every prop must do one of these jobs:
   - **Reveal:** Show what is hidden
   - **Simplify:** Make a complex idea instantly graspable
   - **Contrast:** Show the difference between two choices
   - **Prove:** Demonstrate the principle in real time
   - **Anchor:** Give the audience something they remember later

2. **One hero prop per script.** Multiple props can appear, but one should carry the weight of the argument. The audience should remember that prop when they think of the episode.

3. **Props evolve across the body of work.** The mirror that appears in Script 1 should reappear in Script 7 (visibility/exposure) and Script 9 (silence/assumptions) — each time carrying accumulated meaning.

4. **Build the physical kit.** Own these objects. Bring them to every shoot. They become part of the visual language the same way a recurring set piece becomes part of a TV show's identity.
